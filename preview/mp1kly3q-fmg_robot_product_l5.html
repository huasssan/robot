### 2. Frame Tree

页面 Frame Tree 建议功能位置 关键功能细节
Global Shell Shell -> Top Context Bar -> Side Navigation -> Main Workspace -> Right Attention Rail -> Toast/Modal Layer 顶部放 site/shift/scenario；左侧放一级导航；右侧放全局 attention 和 mission context。 显示当前 mock 场景、数据更新时间、当前用户角色；全局 E-stop 只在任务执行相关页面激活；任何页面可跳转到当前 mission。
AI Operations Command Center Page -> KPI Strip -> Operations Map Context -> Events & Attention -> Fleet Snapshot -> Production/Safety Summary -> Recent Evidence 顶部放 KPI；中间主区域放 remote operations map/context；右侧放 attention queue；底部放 evidence 和 fleet 摘要。 事件列表支持按 risk/site/asset 筛选；点击 CV-12 anomaly 打开 Mission 或 Anomaly Detail；KPI 使用 FY25 shipments、The Hive、CAS 等公开资料校准的 mock 指标。
Mission Orchestration Center Page -> Mission Header -> Mission Graph -> Route Plan Panel -> AI Reasoning Panel -> Safety Gate Panel -> Edge Runtime Panel -> Robot Match Panel -> HITL Drawer -> Execution Timeline 中间放 mission graph；靠近图谱放 route plan 和 local trajectory 状态；右侧放 AI explanation、safety gate、edge runtime、robot match；底部放 execution timeline。 mission graph 节点可选中；每个 step 展示 required skill、assigned robot、status、verification required；route plan 展示 map version、global path、collision risk、fallback policy；审批 drawer 展示 approve/recalculate/hold/e-stop。
Spatial Intelligence Page -> Spatial Context Header -> 3D Semantic Map -> Layer Status -> Global Route Plan -> Local Trajectory & Collision Risk -> Zone Policy Panel -> Safe Corridor Alternatives -> Action Bar 主区域放 3D semantic map 和 path overlay；左侧或上方放 layer/zone filters；右侧放路径、冲突解释和安全动作。 展示 point cloud、occupancy、ESDF、semantic labels、blast zone、truck route、CAS 20m exclusion、restricted corridor；每条冲突说明来源、影响 mission、推荐动作、fallback policy。
Inspection / Anomaly Detail Page -> Anomaly Header -> Signal Evidence Tabs -> AI Interpretation -> Production Impact -> Required Skills -> Mission Link 顶部放异常摘要；中部放 acoustic/thermal/visual tab；右侧放 AI 解释和创建 mission 动作。 声学频段、热异常、视觉摘要均为 mock；显示 confidence、severity、estimated loss、missing context；可进入 Mission Orchestration。
Emergency Intervention Detail Page -> Intervention Header -> Physical Step Checklist -> Robot Status Stack -> Edge Runtime State -> Collision Events -> Safety Gate -> Sensor Feed Mock -> Verification Panel 左侧放步骤 checklist；中间放 sensor feed/mock evidence；右侧放 safety gate、edge runtime、collision events 和 E-stop。 Humanoid intervention 默认 blocked by approval；inspection steps 可先执行；物理干预需展示 required role、审批状态、edge offline policy、local avoidance action 和 emergency stop。
Skill Registry Page -> Skill Category Filter -> Skill List -> Skill Detail -> Compatibility Matrix -> Evidence Requirement -> Standards Reference 左侧放 skill taxonomy；中间放 detail；右侧放 robot compatibility 和 evidence required。 支持 Thermal Inspection、Acoustic Verification、Safe Corridor Navigation、Manual Intervention；展示 payload、minimum readiness、safety level、VDA 5050/MassRobotics/ISO reference。
Robot Capability Profile Page -> Robot Header -> Current Mission -> Payload & Adapter -> Edge Node & Autonomy Mode -> Capability Radar -> PHM Alerts -> Skill Readiness -> Mission History 顶部放 identity/status；中部放 capability、payload 和 edge node；右侧放 PHM、current mission、quick actions。 展示 battery、SoH、connectivity、edge_node_id、autonomy_mode、local_policy_version、PC/HG、adapter protocol、interop profile；PHM task_blocking 或 edge policy mismatch 会直接影响 mission match。
Capability Evaluation Page -> Scenario Selector -> Seven Required Dimensions -> Weight Calculator -> PC/HG Result -> Screening Decision -> Data Import Policy -> Improvement Notes 顶部放 scenario selector 和评测对象；中间放七维必评表和权重计算；右侧放 PC/HG、处置策略和数据入库策略。 按材料要求展示 Operation Types、Repetitiveness、Working Range、SKU Variation、Perception & Visual Judgement、Ergonomics、Environment；每维 1-5 分，自动计算 weighted score、PC/HG 和 reject/pilot/full pass。
Visual Data Intelligence Page -> Evidence Search -> Evidence Cards -> Geo Index Panel -> Timeline Replay -> Dataset Readiness -> Training Gaps 左侧放筛选；中间放 evidence cards 和 timeline；右侧放 dataset readiness 和 gaps。 按 mission、asset、evidence quality、dataset tags 筛选；点击 evidence 回到 Mission replay；显示 training readiness 但标注 mock。

  [D2 Repetitiveness: Low/Medium/High, score 1-5]

  [D4 SKU Variation: Fixed/Slight/Heavy Dynamic, score 1-5]

  Repetitiveness 15%
  Working Range 20%
  SKU Variation 15%
  Perception & Vision 10%
  Comprehensive Environment Margin 25%

  1.0-2.4 -> PC1/HG1 -> Reject, archive only
  2.5-3.9 -> PC2/HG2 -> Pilot, partial data import
  4.0-5.0 -> PC3/HG3 -> Full pass, priority data import
